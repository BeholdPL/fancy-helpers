== Description ==

A versatile WordPress plugin with various helper functions and customizations.

== Features ==

* No orpahns in paragraphs and headings
* Lightbox
* Additional field for H1
* Unify Rank Math icon in Gutenberg
* Showing Patterns menu in Dashboard and disabling default patterns
* Using Site Logo on the login page
* Removing versions from scripts
* Disabling theme editor for classic themes
* Quick button for editing the Home Page
* Fix GenerateBlocks styles in Blocksy Theme
* Global Contacts for easy access (hours, addresses, phones, emails, social media, customs)
* Removing orphans from text
* Implementation of smart Template Tags

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/ directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to Settings > Fancy Helpers to see and use the options.
